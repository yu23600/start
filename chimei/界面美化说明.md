- 文字清晰可读

## 🔧 技术细节

### Canvas渐变实现
```python
for i in range(70):
    color_ratio = i / 70
    r = int(102 + (118 - 102) * color_ratio)
    g = int(126 + (75 - 126) * color_ratio)
    b = int(234 + (162 - 234) * color_ratio)
    color = f'#{r:02x}{g:02x}{b:02x}'
    canvas.create_line(0, i, 800, i, fill=color)
```

### 样式系统
- 使用ttk.Style统一管理
- 支持悬停(hover)和按下(pressed)状态
- 自定义布局确保文字显示

### 响应式设计
- 最小尺寸限制 (minsize)
- 支持窗口调整 (resizable=True)
- 内容自动适应
