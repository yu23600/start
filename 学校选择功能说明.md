# 🏫 学校选择功能使用指南

## 📋 功能概述

新版"吃了么"采用了更简洁的学校管理方式:
- ✅ **首次使用向导**: 首次启动时弹出学校选择界面
- ✅ **274所高校名单**: 来自教育部官网的权威数据
- ✅ **智能搜索**: 快速查找您的学校
- ✅ **一键修改**: 随时更换学校
- ✅ **记忆功能**: 自动记住上次选择的学校

---

## 🎯 使用流程

### 1️⃣ 首次使用

当您第一次访问应用时,会看到欢迎向导:

```
🎉 欢迎使用"吃了么"
首次使用,请选择您的学校
```

**三种选择方式:**

#### 方式A: 从列表中选择
1. 浏览滚动的高校列表(274所)
2. 点击您所在的学校
3. 点击"确认选择"按钮

#### 方式B: 搜索学校
1. 在搜索框输入学校名称
2. 系统实时过滤匹配的学校
3. 点击选中的学校
4. 点击"确认选择"按钮

#### 方式C: 手动输入
1. 如果列表中没有您的学校
2. 在"或者手动输入"框中输入校名
3. 点击"确认选择"按钮

---

### 2️⃣ 正常使用

选择学校后,页面顶部会显示:

```
🏫 当前学校: [您的学校名]    [✏️ 修改学校]
```

- 左侧显示当前学校名称
- 右侧有"修改学校"按钮

---

### 3️⃣ 修改学校

如需更换学校:

1. 点击 **"✏️ 修改学校"** 按钮
2. 确认提示:"修改学校将清空当前数据,确定要继续吗?"
3. 点击"确定"后页面刷新
4. 重新显示学校选择向导
5. 选择新学校

⚠️ **注意**: 修改学校会清除localStorage记录,但不会删除本地数据文件

## 💡 常见问题

### Q1: 为什么我的学校不在列表中?

**原因**: 列表只包含274所主要本科院校

**解决方法**: 
- 使用"手动输入"功能
- 直接输入您的学校名称即可

### Q2: 修改学校后,之前的数据还在吗?

**答案**: 是的!

- 本地数据文件(`cafeteria_data.json`)不会被删除
- 只是清除了"当前学校"的记忆
- 您可以随时通过云端同步恢复数据

### Q3: 如何彻底重置应用?

**步骤**:
1. 清除浏览器localStorage
2. 删除 `cafeteria_data.json` 文件
3. 刷新页面

### Q4: 可以添加多个学校吗?

**答案**: 可以!

虽然界面只显示当前学校,但后端支持多学校数据:
- 每次修改学校会创建新学校记录
- 所有学校数据都保存在同一文件中
- 可以通过云端同步管理多个学校

### Q5: 大学名单如何更新?

**方法**:
```bash
# 运行更新脚本
python fetch_universities.py

# 会生成最新的universities.json
```
