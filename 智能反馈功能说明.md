# 🎯 智能反馈功能使用说明

## 问题场景

你之前提出的问题非常实际：
> "用户也不知道有没有供应怎么办？"

确实，很多时候我们：
- ❌ 去食堂前不知道今天有哪些菜
- ❌ 看了菜单但不知道哪些菜今天真的做
- ❌ 设置了今日供应但可能不准确

## ✨ 解决方案：边推荐边学习

现在系统采用了**更智能的方式**：不需要提前知道，而是在推荐过程中学习！

### 核心思路

```
传统方式（需要预先知道）:
1. 查看食堂菜单 → 2. 设置今日供应 → 3. 开始推荐

智能方式（边用边学）:
1. 直接开始推荐 → 2. 遇到没有的就标记 → 3. 系统自动学习
```

---

## 📱 使用方法

### 1️⃣ 直接开始推荐

不需要提前设置任何东西，直接点击：
- **随机推荐一个菜品** (简洁模式)
- **帮我决定今天吃什么** (专业模式)

### 2️⃣ 看到推荐结果

弹出的对话框现在有**4个按钮**：

```
┌─────────────────────────────┐
│   为你推荐：                 │
│                              │
│   ✅ 红烧肉                  │
│                              │
│   如果这个菜今天没有，       │
│   请点击"今天没有"按钮       │
│                              │
│  [就这个了！✅] [换一个🔄]  │
│  [今天没有 ❌]   [取消 😅]  │
└─────────────────────────────┘
```

### 3️⃣ 根据实际情况选择

#### 情况A：有这个菜 ✅
- 点击 **"就这个了！"**
- 完成推荐

#### 情况B：想看看其他选项 🔄
- 点击 **"换一个"**
- 系统会推荐另一个菜

#### 情况C：这个菜今天没有 ❌ **新功能！**
- 点击 **"今天没有"**
- 系统会：
  1. 记住这个菜今天没有
  2. 跳过这个菜，推荐下一个
  3. 最后询问是否保存到"今日供应"

#### 情况D：不想吃了 😅
- 点击 **"取消"**
- 退出推荐

## 🧠 智能学习机制

### 工作流程

```
推荐第1个菜: 红烧肉
  → 用户点击"今天没有"
  → 系统记录：红烧肉(今天没有)
  → 继续推荐...

推荐第2个菜: 鱼香肉丝
  → 用户点击"就这个了"
  → 系统弹出：
    "你刚才标记了1个菜品今天没有：红烧肉
     是否自动将这些菜品从'今日供应'中移除？"
  
  → 用户点击"是"
  → 系统自动更新今日供应列表
  → 下次不会再推荐红烧肉
```

### 批量学习

如果你在一次推荐过程中标记了多个菜：

```
标记的菜品：
- 红烧肉 (今天没有)
- 麻婆豆腐 (今天没有)
- 清炒时蔬 (今天没有)

接受推荐后，系统会问：
"你刚才标记了3个菜品今天没有：
 红烧肉, 麻婆豆腐, 清炒时蔬

是否自动将这些菜品从'今日供应'中移除？"
```

  2. 设置确定的菜品（比如主食、常备菜）

## 🔧 技术细节

### 数据存储

标记的"今天没有"的菜品会：
1. 临时保存在内存中（本次推荐会话）
2. 用户确认后写入 `cafeteria_data.json`
3. 更新 `available_items` 列表

### 智能提示

系统只在以下情况提示保存：
- ✅ 用户接受了某个推荐
- ✅ 且之前标记过"今天没有"的菜品

如果用户取消或一直换，不会提示。

### 自动初始化

如果之前没有设置过"今日供应"：
- 第一次标记"今天没有"时
- 系统会自动将所有菜品设为"今日供应"
- 然后移除标记为"没有"的菜品

## 💬 常见问题

### Q1: 我不小心点错了"今天没有"怎么办？
**A**: 没关系！你可以：
- 重新打开"设置今日供应"
- 把这个菜重新勾选上
- 或者等明天，今日供应会重置

### Q2: 每天都要重新设置吗？
**A**: 有三种选择：
- **每天设置**: 最准确，适合食堂每天都换菜单
- **偶尔设置**: 平衡准确性和便利性
- **不设置**: 让系统自己学习，适合菜单相对固定

### Q3: 标记"今天没有"后，明天还会推荐吗？
**A**: 会的！"今日供应"只影响当天：
- 今天标记"没有" → 今天不再推荐
- 明天默认恢复（除非你再次设置）

### Q4: 能不能永久删除某个菜？
**A**: 可以！在菜单编辑中：
- 直接删除那行
- 保存菜单
- 这个菜就永远不会被推荐了
